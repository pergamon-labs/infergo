# Benchmarks

This directory is for local maintainer benchmark artifacts and focused
CPU-first benchmark notes.

Checked-in benchmark code lives under:

- `infer/packs`
- `infer/httpserver`

Local raw outputs generated by `./scripts/benchmark_snapshot.sh` should live
under `./benchmarks/local/` and stay gitignored.

InferGo only keeps summarized benchmark guidance in the public repo. For the
current workflow, see:

- [`../BENCHMARKS.md`](../BENCHMARKS.md)

The benchmark suite focuses on:

- model load overhead
- core inference latency
- allocation behavior
- parity smoke-path overhead where relevant
