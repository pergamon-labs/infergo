# InferGo v0.1.0-prealpha.1

InferGo is a Go-native inference and model-serving toolkit for backend
services.

This first public pre-alpha release is intentionally narrow. It focuses on a
small, honest, CPU-first serving surface that we can validate end to end in Go.

## What is included

- stable public APIs in `infer` for loading text and token classifiers
- curated checked-in pack helpers in `infer/packs`
- parity tooling in `cmd/infergo-parity`
- pack discovery in `cmd/infergo-packs`
- runnable text and token HTTP examples
- public-safe native text-classification packs
- public-safe native token-classification packs
- first raw-text-capable native text pack: `infergo-basic-sst2`
- first raw-text-capable native token pack: `infergo-basic-french-ner`

## Benchmark snapshot

Example snapshot from one local run on `darwin/arm64` with an Apple M3 Max:

- `BenchmarkLoadTextPackInfergoBasicSST2`: about `0.34 ms/op`
- `BenchmarkPredictTextInfergoBasicSST2`: about `1.6 µs/op`
- `BenchmarkLoadTokenPackInfergoBasicFrenchNER`: about `0.61 ms/op`
- `BenchmarkPredictTextInfergoBasicFrenchNER`: about `7.3 µs/op`
- `BenchmarkPredictTokensInfergoBasicFrenchNER`: about `5.8 µs/op`

These numbers are illustrative, not a promise. Use
[`BENCHMARKS.md`](../../BENCHMARKS.md) to reproduce them on your own hardware.

## Known limitations

- support is limited to explicitly documented native bundles and curated packs
- raw-text support is intentionally narrow
- TorchScript remains an optional bridge, not the main story
- InferGo is not a blanket `.pt` loader, general transformer runtime, or
  training framework

## Recommended getting-started path

1. List the curated checked-in packs:

```bash
go run ./cmd/infergo-packs
```

2. Run the test suite:

```bash
go test ./...
```

3. Run a parity check:

```bash
go run ./cmd/infergo-parity \
  -reference ./testdata/reference/token-classification/infergo-basic-french-ner-reference.json \
  -infergo-bundle-dir ./testdata/native/token-classification/infergo-basic-french-ner-windowed-embedding-linear \
  -tolerance 1e-3
```

4. Run the token HTTP example:

```bash
go run ./examples/token-http-server
```
