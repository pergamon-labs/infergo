# Code of Conduct

## Our Commitment

We want InferGo to be a welcoming, safe, and constructive project for people of
all backgrounds and experience levels.

Participants in this project are expected to:

- be respectful and considerate in communication
- assume good intent and respond with curiosity
- give actionable, technical feedback without hostility
- avoid harassment, discrimination, and personal attacks
- keep discussions focused on improving the project

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment, intimidation, or threats
- hateful, discriminatory, or demeaning language
- sexualized language or unwanted attention
- doxxing or sharing private information without consent
- sustained disruption, trolling, or bad-faith engagement
- publishing someone else's private data, credentials, or confidential material

## Scope

This Code of Conduct applies in:

- GitHub issues, pull requests, discussions, and reviews
- project chat or coordination spaces
- public or private communication when someone is representing the project

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull
requests, or other contributions that violate this Code of Conduct.

Maintainers may also take temporary or permanent moderation actions when needed
to protect the project and its contributors.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please report it privately to the maintainers through the repository contact
channel or private maintainer outreach.

When reporting, include:

- what happened
- where it happened
- links or screenshots if relevant
- any context that would help review the issue

We will review reports in good faith and try to respond with care and
confidentiality.
